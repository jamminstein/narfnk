# narfnk

> Quad-channel stochastic MIDI funk machine

## Controls

- **E1** — select step (hold K1 to select track)
- **E2** — select parameter
- **E3** — adjust parameter
- **K1+K3** — global start / stop
- **K3** — tap=randomize step, hold=save to slot

## Grid

None required.

## Requirements

- norns
- MIDI device (4 channels)

## Install

```
;install https://github.com/jamminstein/narfnk
```
